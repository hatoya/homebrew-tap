# hatoya/homebrew-tap

Homebrew tap for [hatoya](https://github.com/hatoya)'s apps.

## Usage

```bash
brew install --cask hatoya/tap/ccglance
```

## Casks

| Cask | Description |
| --- | --- |
| [ccglance](https://github.com/hatoya/ccglance) | Always-on-top floating panel that shows Claude Code session status (Apple Silicon, macOS 12+) |

Cask versions are updated automatically by the [ccglance release workflow](https://github.com/hatoya/ccglance/blob/main/.github/workflows/release.yml).
